# FFMall
## 项目介绍
商城系统

## 项目内容
### 组织结构
```lua
mall
├── FFMallDocumentLibrary --文档库
├── FFMall-BUP --buyer user portal 买家用户门户
├── FFMall-SUP --seller user portal 卖家用户门户
├── FFMall-OP --operater portal 操作员门户
├── FFMall-AP --administrators portal 管理员门户
├── FFMall-COMMON --公共模块
├── FFMall-PM --product mamagement 产品管理模块
├── FFMall-SM --system mamagement 系统管理模块
├── FFMall-SC --search center 搜索中心模块
├── FFMall-IFBUP --interface UBP 商城前台接口模块
├── FFMall-IFSUP --interface UMP 商家管理接口模块
├── FFMall-IFOP --interface OP 操作员后台接口模块
├── FFMall-IFAP --interface AP 操作员后台接口模块
├── FFMall-IFIN --interface in 外围调用商城接口模块
├── FFMall-IFOUT --interface out 商城调用外围接口模块
├── FFMall-TOOLS --商城独立工具 
└── FFMall-DEMO --测试代码模块
```
